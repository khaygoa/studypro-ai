# 📚 StudyPro AI

## 🚀 Giới thiệu

StudyPro AI là ứng dụng hỗ trợ học tập sử dụng AI giúp học sinh:

* Theo dõi thời gian tự học
* Phân tích hiệu suất học tập
* Đưa ra gợi ý cải thiện

## 📊 Công nghệ sử dụng

* Python (Tkinter GUI)
* Excel Dataset
* Thuật toán thống kê (Mean, Median, Mode)

## ⚙️ Cách chạy

### Cách 1: Chạy Python

```bash
python studypro_gui.py
```

### Cách 2: File .exe

Chạy file:

```
studypro_gui.exe
```

## 📈 Tính năng

* Nhập dữ liệu học tập
* Phân tích kết quả
* Gợi ý học tập thông minh
